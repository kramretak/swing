3.3 - Buttons

Buttons can be either <a>, <input> or <button> elements, as long as they have a .btn class.

3.5 - Images

By default, images have no border. Optionally, a grey border can be added by using the .border-grey class. Additional class could be added later.

Default styling:

Preview
HTML
    <img src="http://placehold.it/100x100/FF6600/FFFFFF&text=img" width="100" height="100" alt="No border" class="{$modifiers}" />

Add a grey border around the image.

.border-grey

3.6 - Lists

By default, lists will be stripped from their bullets, to make them easier to work with. Alternatively, you can add an "inline" class to make the list appear inline (instead of top to bottom) or add a "bulleted" class to force the bullets in front of the list items.

Default styling:

Preview
  • Option A
  • Option B
  • Option C
HTML
  <ul class="{$modifiers}">
    <li>Option A</li>
    <li>Option B</li>
    <li>Option C</li>
  </ul>
