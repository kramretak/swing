2 - Utility classes

2.1 - Grids

The grid system is a collection of mixins that allow for an easy grid layout, divided into 24 virtual columns, which will be floated side by side. In almost all cases, you should aim for having a total of 24 columns in your grid layout. For example: col(12) translates to a column with a width of 50% of its container.

  • col($n)Define the width of your element, by passing this the number of desired columns.
  • col-right($n)Same as col(), but the column will be floated to the right instead of the default left.
  • width($n)Same as col(), but the column will be not be floated at all.
  • prepend($n)Prepend n blank columns.
  • append($n)Append n blank columns.

2.2 - Mixins

util/_mixins.scss contains mixins, mostly around newer CSS3 properties. Use them by prefixing the mixin with "@include" (e.g. @include background-gradient(color1, color2) )

  • transparent($color, $alpha) - Transparent background, 2nd param = decimal transparency level
  • background-gradient($a, $b) - Top-down background gradient
  • background-gradient-ltr($a, $b) - Background gradient, left to right
  • triple-gradient($a, $b, $c) - Top-down background gradient with 3 colors
  • opacity($n) - Apply cross-browser 'opacity' style
  • border-radius($px) - Apply cross-browser 'border-radius'
  • box-shadow($px) - Apply cross-browser 'box-shadow'
  • text-rotate($deg) - Rotate text, param = degrees
  • transition($transition) - CSS3 Transition
  • transform ($transform) - CSS3 Transform

2.3 - Miscellaneous

util/_misc.scss contains helper classes that are commonly used:

  • .clearfix - apply this to the parent container of floated elements, in order to clear the float.
  • .clear - apply this to clear a previously floated sibling element.
  • .hide - applies "display: none"
  • .left - float an element to the left
  • .right - float an element to the right
  • .center - center position an element (note, this is NOT the same as .txt-center)
  • .relative - apply 'position:relative' to an element
  • .padded - apply 2% padding to an element
  • .visible-phone - Show an element ONLY on a smartphone-wide device
  • .visible-tablet - Show an element ONLY on a tablet-wide device
  • .hidden-tablet - Hide an element on tablet-wide devices and larger
  • .hidden-desktop - Hide an element on desktop-wide devices and larger
  • .visible-desktop - Show an element ONLY on a desktop-wide device
