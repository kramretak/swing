1 - Variable definitions

1.1 - Application Settings

The CSS is built "responsively", meaning that it will behave differently on different screen sizes, using the same codebase. In order to keep the different behavior across devices / screen sizes maintainable, the CSS is divided up into 5 screen size "tiers":

  • $page-width-smartphoneAll screens with a width up to 360px
  • $page-width-smartphone-landscapeSmartphones in Landscape mode - Screens up to 480px
  • $page-width-mini-tabletLarge smartphones and Mini Tablets - Up to 568px
  • $page-width-tabletTablets Screens up to 768px
  • $page-width-desktopTablets in Landscape and everything Desktop sized, 960px and up
  • $grid-total-columnsSet to 24 columns
  • $animation-transition-timeBy default, CSS transitions will run for 0.3 seconds.
  • $page-font-familyDefault font family, set to "Trebuchet", "Lucida Grande"
  • $page-font-family-altAlternative font family, set to "Arial", "Helvetica Neue"
  • $page-font-sizeDefault font size, set to 14px.
